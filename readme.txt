# Service 2: Weather Forecast of Singapore

Tell the weather forecast of Singapore for the next 2 hour. User is able to choose/enter a valid location in Singapore to know its weather forecast. 

You may use the API found in (https://data.gov.sg/dataset/weather-forecast?resource_id=571ef5fb-ed31-48b2-85c9-61677de42ca9).

Can be accessed on desktop systems with a modern web browser (e.g. Google Chrome).

## Getting Started

To run the application locally, please perform the following steps
1. Clone the repository (`git clone https://gitlab.com/pengjm18/weatherforecastsg.git`).
2. Load the application by opening `index.html` file in your browser.
3. Enter a valid address in Singapore (e.g. 303a Anchorvale Link).
